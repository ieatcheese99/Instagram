# Instagram

A simple Instagram clone built using Kotlin.

## Fitur Utama

- **Unggah foto**: Pengguna dapat mengunggah foto ke profil mereka.
- **Feed utama**: Tampilkan foto-foto terbaru dari semua pengguna.
- **Sukai dan komentar**: Berikan tanda suka dan komentar pada postingan.
- **Pencarian pengguna**: Cari pengguna lain berdasarkan nama.

## Teknologi yang Digunakan

- **Bahasa pemrograman**: [Kotlin (100%)](https://kotlinlang.org/)
- **Android SDK**: Untuk pengembangan aplikasi Android native.
- **Material Design**: Untuk tampilan antarmuka yang modern.

## Contoh Penggunaan

1. **Clone repositori**
    ```bash
    git clone https://github.com/ieatcheese99/Instagram.git
    ```

2. **Buka di Android Studio**
    - File > Open > Pilih folder `Instagram`

3. **Build dan jalankan aplikasi**
    - Hubungkan perangkat Android / emulator
    - Klik tombol Run
